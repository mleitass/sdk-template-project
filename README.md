# E-commerce Starter

## Services
- **Frontend**: React 19.2.0 + Vite 7.2.4 + Tailwind CSS 4.1.17 (Port 5173)
- **Products**: PHP Slim 4.15.1 (Port 8082)
- **Users**: Python FastAPI 0.120.0 (Port 8000)
- **Database**: PostgreSQL 18.1 (Port 5432)

## Quick Start
1. **Configure**: `cp .env.example .env`
2. **Run**: `docker-compose up --build`

## Structure
- `frontend/`: UI application
- `products-service/`: Product catalog API
- `users-service/`: User management API
