# E-commerce Starter

## Services
- **Frontend**: React + Vite + Tailwind (Port 5173)
- **Products**: PHP Slim 4 (Port 8082)
- **Users**: Python FastAPI (Port 8000)
- **Database**: PostgreSQL (Port 5432)

## Quick Start
1. **Configure**: `cp .env.example .env`
2. **Run**: `docker-compose up --build`

## Structure
- `frontend/`: UI application
- `products-service/`: Product catalog API
- `users-service/`: User management API
